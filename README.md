# 📝 Word Association Test (WAT) - Android App

Welcome to the **Word Association Test (WAT) Android Application**, a digital tool designed to facilitate a key psychological assessment used to evaluate personality traits, cognitive abilities, and underlying thought patterns. This app provides a simple and efficient platform for presenting words and recording the immediate thoughts of a candidate, making it ideal for psychological evaluations, research, and selection processes.

## 🎯 Key Features

  * ✔️ **Sequential Word Display**: Presents a curated collection of words to candidates in an orderly fashion.
  * ✔️ **Instant Response Recording**: Captures user responses immediately, ensuring genuine first thoughts are logged.
  * ✔️ **Flexible Input**: Allows candidates to respond with full statements, short phrases, or single keywords.
  * ✔️ **Intuitive Interface**: A clean, user-friendly design that simplifies the testing process for both administrators and candidates.
  * ✔️ **Seamless Integration**: Built to be easily integrated into larger psychological assessment suites or personality evaluation applications.

-----

## 💡 Use Cases

This application is designed for a variety of professional and research settings:

  * **Psychological Assessments**: A core component in clinical psychology for personality and cognitive evaluation.
  * **Recruitment & Selection**: Widely used in military and defense selection processes (e.g., SSB interviews) to assess a candidate's suitability.
  * **Behavioral Research**: A valuable tool for researchers conducting studies on cognitive function and thought association.
  * **Self-Evaluation**: An excellent tool for individuals preparing for selection processes to practice and understand their own cognitive patterns.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have the following installed on your machine:

  * [Android Studio](https://developer.android.com/studio) (latest version recommended)
  * An Android Virtual Device (AVD) set up in Android Studio or a physical Android device.

### Installation & Usage

1.  **Clone the repository**
    ```sh
    git clone https://github.com/your-username/WAT-App.git
    ```
2.  **Open in Android Studio**
      * Launch Android Studio.
      * Select `File > Open` and navigate to the cloned project directory.
3.  **Build the Project**
      * Allow Android Studio to sync and download the necessary Gradle dependencies.
      * Click the `Build > Make Project` menu item.
4.  **Run the Application**
      * Select your target device (either a virtual device or a connected physical device).
      * Click the `Run 'app'` button (the green play icon).

### How to Use the App

1.  Launch the application on your device.
2.  Words will be displayed sequentially on the screen.
3.  Enter your first immediate thought or response in the provided text field.
4.  Click the "Save" or "Next" button to submit your response and proceed to the next word.
5.  All responses are recorded locally for subsequent analysis.
