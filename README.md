📝 Word Association Test (WAT)
📌 Project Overview
The Word Association Test (WAT) is a psychological tool used to evaluate personality traits, cognitive ability, and thought patterns of individuals. In this project, candidates are presented with a series of words and are required to write down the first thought that comes to their mind. The responses can be in the form of full statements, phrases, or a combination of words.

🎯 Features
✔️ Displays a collection of words to candidates
✔️ Records user responses instantly
✔️ Allows responses in statements, phrases, or keywords
✔️ User-friendly and intuitive interface
✔️ Easy to integrate for psychological assessments and personality evaluation applications

💡 Use Cases
Personality assessment in psychological tests

Cognitive ability evaluation

Military and defence selection processes

Behavioral research and studies

🔧 Technologies Used
Programming Language: Java

Framework (IDE): Android Studio

UI Components: RecyclerView, EditText, Buttons, and more

📝 Usage Instructions
Launch the application.

Words will appear one by one or in a list.

Write your first thought/response in the provided field.

Save or submit your response to proceed to the next word.

Results are recorded for further analysis.
